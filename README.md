# GIPHY React App
A simple gifs search app made with React using GIPHY API

# Requirements
- GIPHY API key

# Technologies
- __React 16__
- __React router 4__
- __Redux__
- __Bootstrap 4__
- __Reactstrap__ for Bootstrap 4 (instead of React-Bootstrap)
